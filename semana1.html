Objetivo de introduzir os estudos ao JavaScript avançado, revisando conteúdos que serão úteis ao longo do curso

Arrays

Um array é um tipo de variável especial ou objeto que permite o armazenamento e estruturação de um conjunto de dados em um único nome, ou seja é uma variável especial que pode conter mais de um valor.

As Arrays são muito utilizadas para representar listas de itens.

Sua sintaxe:

                
                    let lista = [item1, item2, item3];

Objetos

O objeto pode ser visto como um acervo (coleção) de propriedades, e estas propriedades são definidas como uma sintaxe de par chave:valor, ou seja,a propriedade é uma associação entre um nome (ou chave) e um valor, onde chave pode ser uma string enquanto que o valor pode ser uma informação qualquer ou pode ser uma função, que é então considerada um método do objeto.

Sua sintaxe:

                
                    let object_name(parâmetros)  = {chave: valor1, chave: valor2, ... }

                    ou

                    let object_name(parâmetros)  = {
                        chave: valor1, 
                        chave: valor2, 
                        ... 
                    }

Funções

Em linguagens de programação as funções podem ser definidas como subconjuntos de linhas de códigos, como se fosse um subprograma, que pode ser chamado por outro, ou seja, assim como nos laços de repetição, que nos permite ter blocos de código que vão ser repetidos, nas funções também podem podemos fazer blocos, que serão utilizados em diferentes locais do código e até mesmo repetidas vezes. Além disso, as funções podem receber argumentos, ou seja, receber dados para usar dentro dos blocos de código.

Sua sintaxe:

                
                    function  nome_da_funcao(parâmetros) = {
                        return  parâmetro; 
                    }
